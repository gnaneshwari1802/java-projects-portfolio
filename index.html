Project 1

The Tea Shop Billing System is a standalone desktop application developed using Java SE. It is designed to streamline the billing process in a tea shop, allowing for efficient management of customer orders, bill generation, and sales tracking. The application provides a user-friendly interface for shop employees to handle day-to-day transactions seamlessly.

Project 2

The Pizza Delivery Project is a Java-based system that simulates ordering and delivering pizzas, featuring classes for pizzas, customers, orders, and delivery management. It allows users to place orders, manage delivery statuses, and calculate total prices. The system includes a main driver to demonstrate functionality through a simple command-line interface.
